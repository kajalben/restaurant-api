Mini Yelp Restaurent API

Endpoint for Restarant

GET all Restaurent :::::

https://yelp-restaurant-api.herokuapp.com/restaurant

GET one restaurent by Id :::::

https://yelp-restaurant-api.herokuapp.com/restaurant/:id

GET Restarants by City name :::::

https://yelp-restaurant-api.herokuapp.com/city/restaurants/:name

GET Restarants by Tag name :::::

https://yelp-restaurant-api.herokuapp.com/tag/restaurants/:name

POST create Restaurent :::::

Field : name, image, description, city, tags, location

https://yelp-restaurant-api.herokuapp.com/restaurant
